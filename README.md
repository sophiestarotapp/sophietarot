**Sophie's Tarot** is an AI fantasy companion experience: a living magical parlour where Sophie — a warm, gently mischievous elf tarot reader — gives readings that remember you, tells interactive stories, and grows closer to you with every visit.

## ✨ Features

- **The Parlour (Home)** — Sophie sits at her tarot table in a fully animated magical room: candle flicker, drifting particles, twinkling stars, bookshelves, crystals, and a real day/night cycle driven by your clock.
- **Living Sophie** — hand-built animated SVG character: breathing, blinking, hair sway, head tilt, eye tracking (her eyes follow your cursor), emotional expressions, and her signature big bow. Every style re-themes her outfit, hair accessory, and the room's glow.
- **Tarot Readings** — Daily, Love, Career, Shadow, Past·Present·Future, Celtic Cross, and Custom spreads. Full flow: question → animated shuffle → pick from the fan → 3D card flips → Sophie's interpretation.
- **AI with Memory** — readings are generated by OpenAI with a memory-aware prompt (long-term / recent / emotional / ritual memories + reading history). Without an API key, a built-in local interpretation engine keeps the experience fully playable offline.
- **Novels** — Main Story, Side Stories, Tarot Lore, and Dream Archives with branching choices that raise your Bond and become memories Sophie references later.
- **Collectibles** — tarot cards (collected by drawing them), frames, decorations, stickers, and companions across Common → UR rarities.
- **Boutique** — digital cosmetics (gold/stardust), physical merch & stardust packs (Stripe), and the Moonkeeper membership. Cosmetics only — never pay-to-win.
- **Profile** — reading history, achievements, ritual statistics, collection/story completion, and a relationship timeline built from Sophie's memories of you.
- **Live service hooks** — daily gift with login streaks, daily message, daily reading, events panel.

## 🚀 Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) — the app runs fully offline with local persistence (Zustand + localStorage), no keys required.

### Optional services

Copy `.env.example` to `.env.local` and fill in what you want to enable:

| Service | Enables |
| --- | --- |
| `OPENAI_API_KEY` | True AI-generated readings in Sophie's personality and tone |
| Supabase keys | Auth (Google / Apple / Email), cloud sync, pgvector RAG memory (`supabase/schema.sql`) |
| Stripe keys | Real checkout for stardust packs, membership, physical goods |
| PostHog key | Product analytics |
